# 🎮 坦克大战 (Tank Battle)

经典 FC 坦克大战的 HTML5 复刻版本，纯前端实现，无需任何依赖，打开即玩！

## 🕹️ 游戏截图

游戏画面采用 Canvas 渲染，包含 3 个精心设计的关卡地图。

## 🎯 游戏玩法

- 控制你的绿色坦克，消灭所有红色敌军坦克
- **保护你的基地** 🦅 —— 基地位于地图底部中央，被摧毁则游戏结束
- 摧毁砖墙开辟新路径，钢墙无法摧毁
- 树丛可以隐藏坦克行踪
- 击败敌人有概率掉落道具

## ⌨️ 操作方式

| 按键 | 操作 |
|------|------|
| `↑ ↓ ← →` 或 `WASD` | 移动坦克 |
| `空格键` | 发射子弹 |
| `P` | 暂停/继续 |
| `R` | 重新开始 |

📱 移动端支持触摸操控。

## 🎁 道具系统

- ❤️ **生命** - 增加一条命（最多 5 条）
- ⚡ **加速** - 临时提升移动速度
- 🛡️ **护盾** - 临时无敌
- ❄️ **冰冻** - 减缓所有敌人移动速度

## 🚀 快速开始

1. 克隆仓库：
   ```bash
   git clone https://github.com/YOUR_USERNAME/tank-battle.git
   ```
2. 用浏览器打开 `tank-battle.html`
3. 开始战斗！

或者直接双击 `tank-battle.html` 文件在浏览器中打开。

## 🛠️ 技术栈

- HTML5 Canvas
- 原生 JavaScript (ES6+)
- 纯 CSS3 动画
- 零依赖，单文件

## 📝 关卡

游戏内置 3 个关卡，每关敌人数量和速度递增。通关后可继续挑战！
